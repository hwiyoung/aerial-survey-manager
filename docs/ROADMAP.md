### 4.2 🔲 예정 항목

#### 4.2.1 메인 페이지 네비게이션 (우선순위: 높음)
**현황**: 처리 패널이나 상세 화면에서 메인 페이지로 돌아가는 방법 없음

**개선 방안**:
- 헤더의 플랫폼 로고/이름 클릭 시 메인 페이지로 이동
- URL 파라미터 초기화하여 깨끗한 상태로 복귀

**구현 범위**:
```jsx
// Header 컴포넌트 수정
<div onClick={() => window.location.href = window.location.pathname}>
  🗺️ 항공측량관리
</div>
```

---

#### 4.2.2 처리 패널 접근성 개선 (우선순위: 중간)
**현황**: 처리 패널은 업로드 완료 시에만 접근 가능

**문제점**:
- 재처리가 필요한 경우 접근 불가
- 여러 프로젝트 업로드 후 나중에 일괄 처리 불가
- 처리 상태 모니터링 및 관리 어려움

**개선 방안**: 프로젝트 카드에 액션 버튼 추가
```
┌─────────────────────────────────────┐
│ Project_20260108                    │
│ 상태: 대기                          │
│ [📊 대시보드] [▶️ 처리] [📦 내보내기] │
└─────────────────────────────────────┘
```

**구현 범위**:
- `ProjectItem` 컴포넌트에 처리/대시보드/내보내기 버튼 추가
- 버튼 클릭 시 해당 모드로 전환 및 프로젝트 선택

---

#### 4.2.3 반응형 프로젝트 대시보드 (우선순위: 낮음)
**현황**: 사이드바 너비 320px 고정 (리사이즈 가능), 너비와 관계없이 동일한 정보 표시

**개선 방안**: 너비에 따라 표시 정보 동적 변경

| 너비 범위 | 표시 정보 |
|-----------|----------|
| 좁음 (280~400px) | 프로젝트명, 상태 뱃지 |
| 중간 (400~600px) | + 이미지 수, 생성일, 진행률 바 |
| 넓음 (600px+) | + 썸네일, 처리 파라미터, 빠른 액션 버튼 |

**넓은 패널 예시 (600px+)**:
```
┌───────────────────────────────────────────────────────────────┐
│ [체크] 🖼️  Project_20260108_Survey                   [대기]    │
│         📷 10장  📅 2026-01-08  📐 5cm GSD                    │
│         ══════════════════░░░░░░░░ 60%                       │
│         [▶️ 처리 시작] [📊 상세보기] [🗑️]                      │
└───────────────────────────────────────────────────────────────┘
```

**구현 범위**:
- `Sidebar` 컴포넌트에서 너비 감지 로직 추가
- `ProjectItem` 컴포넌트를 너비에 따라 다른 레이아웃 렌더링
- CSS 미디어 쿼리 또는 ResizeObserver 활용
