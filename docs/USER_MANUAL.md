## 1. 로그인

### 화면
- 아이디/비밀번호 입력 폼

### 사용법
1. 아이디 입력 (기본: `admin`)
2. 비밀번호 입력 (기본: `siqms`)
3. "로그인" 버튼 클릭

## 2. 대시보드

### 화면 구성

#### 상단 헤더
- 시스템 로고 및 명칭
- 로그아웃 버튼

#### 좌측 사이드바
| 항목 | 설명 |
|------|------|
| 프로젝트 목록 | 전체 프로젝트를 상태별로 표시 (완료/진행중/대기/오류) |
| 프로젝트 그룹 | 폴더 구조로 프로젝트 분류 |
| 처리 진행 표시 | 처리 중인 프로젝트의 단계별 진행 상태 실시간 표시 |

#### 상단 통계 카드
| 카드 | 내용 |
|------|------|
| 처리 완료 면적 | 완료된 정사영상의 총 면적 (km²) |
| 프로젝트 현황 | 완료/전체 프로젝트 수 |
| 저장 용량 | 정사영상 + 배경지도 타일 사용량 |

#### 중앙 지도
- 전체 프로젝트의 촬영 영역을 지도에 표시
- 프로젝트 클릭 시 해당 위치로 이동
- 완료된 프로젝트는 정사영상 미리보기 표시

#### 사이드바 프로젝트 항목 표시 정보
- 프로젝트명, 지역, 상태 배지
- 처리 완료일
- 이미지 수, 면적, GSD
- 처리 중일 때: 5단계 진행 표시
  - 이미지 정렬 → 깊이 맵 → DEM 생성 → 정사모자이크 → COG 변환

## 3. 프로젝트 생성

"새 프로젝트" 버튼을 클릭하면 4단계 마법사가 열립니다.

### Step 1: 이미지 선택

| 항목 | 설명 |
|------|------|
| 폴더 선택 | 서버 파일 탐색기에서 이미지 폴더 선택 |
| 지원 형식 | JPG, JPEG, PNG, TIFF |
| 외장하드 | /media, /mnt, /home 경로의 외장 드라이브 자동 감지 |

**사용법:**
1. "폴더 선택" 클릭
2. 서버 파일 탐색기에서 이미지가 있는 폴더로 이동
3. 폴더 선택 → 이미지 수 자동 카운트
4. "다음" 클릭

### Step 2: EO (외부방위) 데이터

| 항목 | 설명 |
|------|------|
| EO 파일 | .txt, .csv, .json 형식 지원 |
| 좌표계 | TM중부, TM서부, TM동부, UTM-K, WGS84 |
| 구분자 | 공백, 탭, 쉼표 |
| 헤더 유무 | 첫 줄 헤더 여부 선택 |
| 컬럼 매핑 | 이미지명, X, Y, Z, Omega, Phi, Kappa 컬럼 지정 |

**사용법:**
1. EO 파일 선택 (Step 1에서 선택한 폴더 경로로 자동 이동)
2. 좌표계, 구분자, 헤더 설정
3. 미리보기 테이블에서 데이터 확인
4. 컬럼 번호 조정 (필요시)
5. "다음" 클릭

### Step 3: 카메라 모델 (IO)

| 항목 | 설명 |
|------|------|
| 카메라 선택 | 등록된 카메라 모델 목록에서 선택 |
| 사용자 정의 | 초점거리, 센서 크기, 픽셀 크기, PPA 직접 입력 |

**사용법:**
1. 드롭다운에서 카메라 모델 선택
2. 또는 "카메라 추가"로 새 모델 등록
3. "다음" 클릭

### Step 4: 확인 및 처리 설정

| 항목 | 설명 |
|------|------|
| 프로젝트명 | 프로젝트 이름 입력/수정 |
| 처리 모드 | 미리보기 / 고속 / 정밀 / 고정밀 |
| 자동 처리 | 체크 시 프로젝트 생성 즉시 처리 시작 |

**처리 모드 설명:**

| 모드 | 품질 | 속도 | 용도 |
|------|------|------|------|
| 미리보기 (Preview) | 낮음 | 빠름 | 빠른 결과 확인용 |
| 고속 (Fast) | 중간 | 빠름 | 일반 작업용 |
| 정밀 (Normal) | 높음 | 보통 | 납품용 (기본값) |
| 고정밀 (High) | 최고 | 느림 | 정밀 납품용 |

**사용법:**
1. 프로젝트명 확인/수정
2. 처리 모드 선택
3. "완료 후 자동 처리" 체크 (야간 무인 운영 시 권장)
4. "완료" 클릭 → 프로젝트 생성

## 4. 처리

### 처리 시작
- 사이드바에서 프로젝트 선택 → "처리 시작" 버튼 클릭
- 또는 프로젝트 생성 시 "자동 처리" 옵션 사용

### 처리 단계 (자동 진행)

| 순서 | 단계 | 설명 |
|------|------|------|
| 1 | 이미지 정렬 | 사진 간 특징점 매칭 및 카메라 위치 보정 |
| 2 | 깊이 맵 생성 | 각 사진의 깊이 정보 계산 |
| 3 | DEM 생성 | 수치표면모델 생성 |
| 4 | 정사모자이크 | 왜곡 보정된 정사영상 생성 |
| 5 | COG 변환 | Cloud Optimized GeoTIFF로 변환 (웹 뷰어용) |

### 처리 중 확인
- 사이드바에서 각 단계별 진행률 실시간 확인 (5초 간격 갱신)
- 단계별 상태 표시: 대기(회색) → 진행 중(파란색) → 완료(녹색)

### 처리 중단
- 프로젝트 선택 → "처리 중단" 버튼
- 확인 다이얼로그에서 "중단" 선택

### 처리 오류 복구
- 전원 차단 등으로 중단된 경우: 서버 재시작 시 자동으로 'error' 상태로 전환
- 재처리: "처리 시작" 버튼으로 다시 시작 가능

## 5. 프로젝트 상세 조회

사이드바에서 프로젝트를 클릭하면 하단에 인스펙터 패널이 열립니다.

### 프로젝트 정보 (좌측 2/3)

| 항목 | 설명 |
|------|------|
| 프로젝트명 | 이름 (클릭하여 수정 가능) |
| 상태 | 완료/진행중/대기/오류 |
| 처리 모드 | 정밀/고속/미리보기/고정밀 |
| 정사영상 면적 | km² 단위 |
| GSD | 지상해상도 (cm/px) |
| 정사영상 용량 | COG 파일 크기 |
| 생성일 | 프로젝트 생성 일시 |
| 처리완료일 | 처리 완료 일시 |
| 처리 소요시간 | 총 소요 시간 |

### 정사영상 미리보기 (우측 1/3)
- COG 삭제 후에도 썸네일로 미리보기 표시
- 썸네일 해상도: 4096px

### EO 포인트 클릭
- 지도에서 EO 마커 클릭 시 해당 이미지 정보 표시
- 썸네일 미생성 시 자동으로 온디맨드 생성 (로딩 스피너 표시)
- 이미지명, GPS 좌표 (X, Y, Z), 카메라 각도 (Omega, Phi, Kappa) 표시

## 6. 지도 기능

### 지도 컨트롤 (우상단)

| 버튼 | 기능 |
|------|------|
| 지도 아이콘 | 배경지도 on/off 토글 (정사영상만 볼 때 유용) |
| 십자 아이콘 | 정사영상 원래 범위로 복귀 |
| 눈 아이콘 | EO 포인트 마커 표시/숨김 |

### 지도 헤더 (상단)

| 버튼 | 기능 |
|------|------|
| 촬영 영역 | 프로젝트별 촬영 범위 폴리곤 표시 |
| 권역 | 전국 권역 경계 표시 |
| 도엽 | 도엽 격자 오버레이 표시 (아래 7장 참조) |

### 지도 조작
- 마우스 드래그: 이동 (자유롭게 패닝 가능)
- 마우스 휠: 확대/축소
- 프로젝트 클릭: 해당 프로젝트 영역으로 이동
- 정사영상: 완료된 프로젝트는 COG 타일로 스트리밍 표시

## 7. 도엽 격자

납품/검수 시 국가 표준 도엽 단위로 정사영상을 분할하는 기능입니다.

### 도엽 격자 켜기
1. 지도 상단 헤더에서 "도엽" 버튼 클릭
2. 좌측에 도엽 컨트롤 패널 표시

### 축척 선택
| 축척 | 도엽 크기 | 용도 |
|------|----------|------|
| 1:50,000 | 넓은 영역 | 광역 개요 |
| 1:25,000 | 중간 영역 | 중간 단위 |
| 1:5,000 | 좁은 영역 | 납품용 (일반) |
| 1:1,000 | 매우 좁은 영역 | 납품용 (정밀) |

> 축척 버튼은 해당 GeoJSON 데이터가 있는 경우에만 활성화됩니다.

### 도엽 선택
- **지도 클릭**: 도엽 영역 직접 클릭하여 선택/해제
- **전체 선택**: "전체 선택" 버튼으로 겹치는 도엽 일괄 선택
- **전체 해제**: "전체 해제" 버튼

### 도엽번호 검색
1. 검색 입력란에 도엽번호 입력 (예: `37806043`)
2. 검색 아이콘 클릭
3. 해당 도엽 위치로 자동 이동 + 선택

### 도엽 클립 내보내기
1. 도엽 선택 (1개 이상)
2. "클립" 버튼 클릭
3. 선택한 도엽 범위로 정사영상이 잘려서 다운로드됨
4. 파일명: `{도엽번호}_{프로젝트명}.tif`

### 줌 레벨 제한
- 1:1,000 → 줌 14 이상에서만 표시
- 1:5,000 → 줌 12 이상에서만 표시
- 최대 200개 도엽까지 표시 (초과 시 경고)

## 8. 내보내기

### 내보내기 열기
- 사이드바에서 "내보내기" 버튼 클릭
- 또는 프로젝트 선택 후 내보내기

### 내보내기 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| 좌표계 | TM중부/서부/동부, UTM-K, WGS84 | TM중부 (EPSG:5186) |
| GSD | 출력 해상도 (cm/px) | 원본 GSD |
| 형식 | GeoTIFF | GeoTIFF |

### 내보내기 진행
1. 옵션 선택 후 "내보내기" 클릭
2. 진행률 표시 (준비 → 처리 → 완료)
3. 완료 시 자동 다운로드

### 내보내기 후 COG 삭제
- 내보내기 완료 후 "원본 삭제 여부" 확인 다이얼로그 표시
- 각 프로젝트별 정사영상 용량 표시
- "삭제" 선택 시 COG 파일 삭제 (썸네일은 보존)
- "보관" 선택 시 COG 유지

## 9. 정사영상 관리

### 정사영상 삭제
- 프로젝트 인스펙터 패널에서 "정사영상 삭제" 버튼 클릭
- 삭제 전 확인 다이얼로그 표시
- 삭제 시: COG 파일 삭제, 썸네일 자동 생성 후 보존
- 삭제 후: 프로젝트 메타데이터(면적, GSD, 좌표 등)는 유지
- 재처리로 복구 가능

### 저장 용량 확인
- 대시보드 상단 "저장 용량" 카드에서 전체 사용량 확인
- 정사영상 용량 + 배경지도 타일 용량 합산
